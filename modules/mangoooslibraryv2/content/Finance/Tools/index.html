Finance tools

A list of finance tools used for quantitative analysis. The about section will lead you to a page on my website.

MATLAB


High-level numerical computing environment used for math, modeling, simulations, algorithm development, and finance (quantitative analysis, optimization, risk modeling). Huge ecosystem of toolboxes.

Link: mathworks.com
About: MATLAB

Price: Paid



OriginLab Origin


Data analysis and graphing software with strong visualization features, curve fitting, and statistics. Often used in scientific research and engineering; less finance-focused but still useful for statistical work.

Link: originlab.com
About: Origin

Price: Paid



Stata


Statistical and econometric software widely used in economics and social sciences for regression, panel data analysis, and time-series modeling.

Link: stata.com
About: Stata

Price: Paid



EViews


Econometrics software focused on time-series analysis, macroeconomic modeling, and forecasting. Often used in academic and professional finance research.

Link: eviews.com
About: EViews

Price: Paid



R


Open-source programming language and statistical environment with extensive packages for financial analysis, portfolio modeling, and econometrics.

Link: r-project.org
About: R

Price: Free



Python


General-purpose programming language heavily used in finance for data analysis, machine learning, quantitative trading, and automation.

Link: python.org
About: Python

Price: Free



Julia


High-performance programming language designed for numerical computing, increasingly used in quantitative finance due to its speed and mathematical syntax.

Link: julialang.org
About: Julia

Price: Free



Excel


Spreadsheet software widely used in finance for modeling, valuation, and analysis. Add-ins like Solver and Analysis ToolPak extend statistical capabilities.

Link: microsoft.com
About: Excel

Price: Paid



GNU Octave


Open-source numerical computing environment largely compatible with MATLAB syntax. Useful for mathematical modeling and financial calculations.

Link: gnu.org
About: GNU Octave

Price: Free



QuantLib


Open-source library for quantitative finance providing tools for derivatives pricing, risk analysis, and financial modeling.

Link: quantlib.org
About: QuantLib

Price: Free



SAS


Enterprise analytics and statistical software used in banking, risk management, fraud detection, and large-scale financial data analysis.

Link: sas.com
About: SAS

Price: Paid



SPSS


Statistical analysis software developed by IBM with a graphical interface for regression, modeling, and data exploration.

Link: ibm.com
About: SPSS

Price: Paid



Tableau


Business intelligence and data visualization platform widely used to create interactive dashboards for financial and business analytics.

Link: tableau.com
About: Tableau

Price: Paid



Power BI


Microsoft business analytics tool used for financial dashboards, data modeling, and reporting.

Link: powerbi.microsoft.com
About: Power BI

Price: Free / Paid



JMP


Interactive statistical discovery software developed by SAS that focuses on exploratory data analysis and visualization.

Link: jmp.com
About: JMP

Price: Paid



Wolfram Mathematica


Advanced computational software for symbolic mathematics, numerical modeling, and algorithmic finance applications.

Link: wolfram.com
About: Mathematica

Price: Paid
